# 研发投入比对系统实现计划

## 一、主要指标年报小程序分析

### 1.1 代码模块结构

| 模块文件                     | 功能描述                |
| ------------------------ | ------------------- |
| `main_indicator.py`      | 主程序入口，核心数据处理类       |
| `ai_service_enhanced.py` | AI服务封装，处理文件上传、数据提取  |
| `database_manager.py`    | 数据库连接池和查询管理         |
| `config.py`              | 统一配置管理（数据库、AI、处理参数） |
| `logger_config.py`       | 日志配置和会话管理           |

### 1.2 核心处理流程

```
1. 扫描目录PDF文件 → 2. 解析文件名(股票代码-发布日期)
       ↓
3. 上传PDF到AI服务 → 4. 使用提示词提取数据
       ↓
5. 查询数据库获取正式数据 → 6. 数据比对
       ↓
7. 生成Excel比对报告 + AI提取日志
```

### 1.3 关键技术特点

1. **并发处理**：使用 `ThreadPoolExecutor` 实现上传和处理并行
2. **速率控制**：AI服务内置自适应速率限制，避免429错误
3. **连接池**：数据库使用连接池管理，提高效率
4. **会话日志**：每次运行生成唯一会话ID，日志文件独立
5. **模块化设计**：配置、日志、数据库、AI服务分离

### 1.4 比对逻辑

* **主键匹配**：通过 `GPDM`(股票代码) + `XXFBRQ`(信息发布日期) + `JZRQ`(截止日期) 匹配记录

* **文件名解析**：格式为 `股票代码-年-月-日.pdf`，如 `000001-2024-03-15.pdf`

* **比对结果**：逐字段比对，输出差异信息

## 二、研发投入业务分析

### 2.1 AI提取字段（来自 prompt\_YFTR.md）

| 字段英文名       | 字段中文名          | 数据类型 | 说明            |
| ----------- | -------------- | ---- | ------------- |
| JZRQ        | 截止日期           | 日期   | 格式：YYYY-MM-DD |
| HBBZ        | 合并标志           | 文本   | "合并" 或 "母公司"  |
| FYHYFTR     | 费用化研发投入(元)     | 金额   | 需转换万元/亿元      |
| ZBHYFTR     | 资本化研发投入(元)     | 金额   | 需转换万元/亿元      |
| YFTRHJ      | 研发投入合计(元)      | 金额   | 需转换万元/亿元      |
| YFTRZYYSRBL | 研发投入占营业收入比例(%) | 比例   | 去除%符号         |
| ZBHYFTRZB   | 资本化研发投入占比(%)   | 比例   | 去除%符号         |
| YFRYSL      | 研发人员数量         | 整数   | -             |
| YFRYSLZB    | 研发人员数量占比(%)    | 比例   | 去除%符号         |

### 2.2 SQL查询字段（来自 研发投入.sql）

```sql
SELECT A.JZRQ,
       CASE WHEN A.HBBZ=1 THEN '合并' ELSE '母公司' END AS HBBZ,
       A.FYHYFTR, A.ZBHYFTR, A.YFTRHJ, 
       A.YFTRZYYSRBL, A.ZBHYFTRZB, 
       A.YFRYSL, A.YFRYSLZB
FROM [10.101.0.212].JYPRIME.dbo.usrGSYFTRYCC A
JOIN [10.101.0.212].JYPRIME.dbo.usrZQZB B 
  ON A.INBBM=B.INBBM AND B.ZQSC IN (18,83,90) AND B.ZQLB IN (1,2,41)
WHERE A.XXLYBM='110101' AND B.GPDM=? AND A.XXFBRQ=?
```

**查询参数**：`GPDM`(股票代码)、`XXFBRQ`(信息发布日期)

### 2.3 比对逻辑

1. **主键**：`GPDM` + `XXFBRQ` + `JZRQ`（用户确认）
2. **匹配规则**：

   * 从文件名解析 `GPDM` 和 `XXFBRQ`

   * 通过SQL查询获取数据库记录

   * AI数据按 `JZRQ` 匹配数据库记录

   * `HBBZ` 作为数据字段参与比对，不作为主键
3. **比对字段**：`HBBZ`, `FYHYFTR`, `ZBHYFTR`, `YFTRHJ`, `YFTRZYYSRBL`, `ZBHYFTRZB`, `YFRYSL`, `YFRYSLZB`

### 2.4 与主要指标的差异

| 对比项   | 主要指标                 | 研发投入                 |
| ----- | -------------------- | -------------------- |
| 模块数量  | 2个模块（模块一+模块二）        | 1个模块                 |
| 主键字段  | GPDM + XXFBRQ + JZRQ | GPDM + XXFBRQ + JZRQ |
| 数据合并  | 需要合并两个模块数据           | 无需合并                 |
| 比对字段数 | 约35个字段               | 8个字段（含HBBZ）          |
| 特殊处理  | 每股收益精度合并             | 金额单位转换               |
