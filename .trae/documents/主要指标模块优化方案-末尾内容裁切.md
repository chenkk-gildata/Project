# 主要指标模块优化方案 - 新增末尾内容裁切

## 问题分析

### 当前状况

* `zyzb_processor.py` 当前搜索范围：`[(4, 10), (9, 20), (19, 30), (29, 50)]`，最多搜索到第50页

* 年报PDF通常200-300页

* 新增需要裁切的内容位于PDF**最后5页以内**

* 如果直接扩展搜索范围，需要从头到尾遍历整个PDF，效率极低

### 新增内容需求

| 项目    | 内容                |
| ----- | ----------------- |
| 位置    | 最后5页以内            |
| 开始关键词 | "净资产收益率及每股收益"     |
| 结束关键词 | "境内外会计准则下会计数据差异"  |
| 搜索关键词 | 每股收益、会计准则         |
| 输出方式  | 追加到主内容后面，合并为一个PDF |

### 特殊处理

* 从后往前搜索，开始和结束关键词可能不在同一页

* 需要允许没找到开始关键词时也搜索结束关键词

***

## 最终方案：反向搜索策略

### 方案架构

```
┌─────────────────────────────────────────────────────────────┐
│                    find_keywords 方法                        │
├─────────────────────────────────────────────────────────────┤
│  阶段一：前段搜索（现有逻辑）                                  │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ 搜索范围: [(4, 10), (9, 20), (19, 30), (29, 50)]    │    │
│  │ 搜索内容: start_info, end_info, remove_start/end    │    │
│  └─────────────────────────────────────────────────────┘    │
│                            ↓                                │
│  阶段二：后段反向搜索（新增逻辑）                              │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ 搜索范围: 最后5页（从后往前）                         │    │
│  │ 搜索内容: append_start_info, append_end_info        │    │
│  └─────────────────────────────────────────────────────┘    │
│                            ↓                                │
│  返回结果: {start, end, ..., append_start, append_end}      │
└─────────────────────────────────────────────────────────────┘
                             ↓
┌─────────────────────────────────────────────────────────────┐
│                    process_pdf 方法                          │
├─────────────────────────────────────────────────────────────┤
│  1. 提取主内容页面（现有逻辑）                                 │

│  3. 合并两个部分，输出单个PDF                                 │

```

## 详细实现步骤

### 步骤1：修改 `find_keywords` 方法

在现有搜索逻辑后，新增反向搜索逻辑：

```python
def find_keywords(self, pdf_path: str) -> Dict:
    """查找主要指标关键词"""
    is_bse = is_bse_pdf(pdf_path)
    
    # ... 现有的前段搜索逻辑保持不变 ...
    
    doc = fitz.open(pdf_path)
    total_pages = len(doc)
    
    # ========== 现有前段搜索逻辑 ==========
    # ... 保持不变 ...
    
    # ========== 新增：后段反向搜索逻辑 ==========
    append_start_info = None
    append_end_info = None
    
    # 定义追加内容的搜索模式
    append_start_pattern = re.compile(r'净资产收益率及每股收益', re.IGNORECASE | re.MULTILINE)
    append_end_pattern = re.compile(r'境内外会计准则下会计数据差异', re.IGNORECASE | re.MULTILINE)
    
    # 反向搜索范围：最后5页
    append_search_start = max(1, total_pages - 5)
    
    # 从后往前搜索
    for page_num in range(total_pages - 1, append_search_start - 1, -1):
        page = doc.load_page(page_num)
        page_rect = page.rect
        
        # 搜索结束关键词（先搜索结束，因为从后往前）
        if not append_end_info:
            instances = page.search_for("会计准则")
            for inst in instances:
                rect = self._get_search_rect(page, inst, "会计准则")
                text = page.get_text("text", clip=rect)
                if append_end_pattern.search(text):
                    append_end_info = {
                        'page_number': page_num + 1,
                        'keyword_box': (inst.x0, inst.y0, inst.x1, inst.y1),
                        'page_dimensions': (page_rect.width, page_rect.height)
                    }
                    break
        
        # 搜索开始关键词
        if not append_start_info:
            instances = page.search_for("每股收益")
            for inst in instances:
                rect = self._get_search_rect(page, inst, "每股收益")
                text = page.get_text("text", clip=rect)
                if append_start_pattern.search(text):
                    append_start_info = {
                        'page_number': page_num + 1,
                        'keyword_box': (inst.x0, inst.y0, inst.x1, inst.y1),
                        'page_dimensions': (page_rect.width, page_rect.height)
                    }
                    break
        
        # 如果都找到了，提前退出
        if append_start_info and append_end_info:
            break
    
    doc.close()
    
    # 构建返回结果
    result = {
        'start': start_info,
        'end': end_info,
        'is_bse': is_bse,
        # 新增追加内容关键词
        'append_start': append_start_info,
        'append_end': append_end_info
    }
    if is_bse:
        result['remove_start'] = remove_start_info
        result['remove_end'] = remove_end_info
    
    return result
```

### 步骤3：新增辅助方法 `_extract_append_pages`

```python
def _extract_append_pages(self, pdf_path: str, append_start_info: Dict, append_end_info: Dict) -> list:
    """
    提取追加内容页面
    
    Args:
        pdf_path: PDF文件路径
        append_start_info: 开始关键词信息
        append_end_info: 结束关键词信息
    
    Returns:
        list: 页面列表
    """
    from PyPDF2 import PdfReader
    
    reader = PdfReader(pdf_path)
    total_pages = len(reader.pages)
    append_pages = []
    
    # 情况1：只找到结束关键词 -> 包含结束关键词所在页 + 往前取一页
    if not append_start_info and append_end_info:
        end_page = append_end_info['page_number']
        # 往前取一页（包含结束关键词所在页）
        start_page = max(1, end_page - 1)
        for i in range(start_page - 1, end_page):
            page = reader.pages[i]
            # 裁剪结束页：保留结束关键词之前的内容
            if i == end_page - 1:
                page_width, page_height = append_end_info['page_dimensions']
                _, _, _, max_y = append_end_info['keyword_box']
                pydf2_max_y = page_height - max_y
                page.cropbox.lower_left = (0, pydf2_max_y)
                page.cropbox.upper_right = (page_width, page_height)
            append_pages.append(page)
    
    # 情况2：只找到开始关键词 -> 包含开始关键词所在页 + 往后取一页
    elif append_start_info and not append_end_info:
        start_page = append_start_info['page_number']
        # 往后取一页（包含开始关键词所在页）
        end_page = min(total_pages, start_page + 1)
        for i in range(start_page - 1, end_page):
            page = reader.pages[i]
            # 裁剪开始页：保留开始关键词之后的内容
            if i == start_page - 1:
                page_width, page_height = append_start_info['page_dimensions']
                _, min_y, _, _ = append_start_info['keyword_box']
                pydf2_max_y = page_height - min_y
                page.cropbox.lower_left = (0, 0)
                page.cropbox.upper_right = (page_width, pydf2_max_y)
            append_pages.append(page)
    
    # 情况3：都找到了
    elif append_start_info and append_end_info:
        start_page = append_start_info['page_number']
        end_page = append_end_info['page_number']
        
        # 情况3.1：开始页 > 结束页 -> 按结束关键词页往前取一页处理
        if start_page > end_page:
            target_page = max(1, end_page - 1)
            append_pages.append(reader.pages[target_page - 1])
        
        # 情况3.2：开始页 == 结束页（同一页）
        elif start_page == end_page:
            page = reader.pages[start_page - 1]
            page_width, page_height = append_start_info['page_dimensions']
            _, start_min_y, _, _ = append_start_info['keyword_box']
            _, _, _, end_max_y = append_end_info['keyword_box']
            
            # 转换y坐标
            pydf2_start_max_y = page_height - start_min_y
            pydf2_end_max_y = page_height - end_max_y
            
            # 裁剪：保留开始关键词之后、结束关键词之前的内容
            page.cropbox.lower_left = (0, pydf2_end_max_y)
            page.cropbox.upper_right = (page_width, pydf2_start_max_y)
            append_pages.append(page)
        
        # 情况3.3：正常情况（开始页 < 结束页）
        else:
            for i in range(start_page - 1, end_page):
                page = reader.pages[i]
                
                # 裁剪开始页：保留开始关键词之后的内容
                if i == start_page - 1:
                    page_width, page_height = append_start_info['page_dimensions']
                    _, min_y, _, _ = append_start_info['keyword_box']
                    pydf2_max_y = page_height - min_y
                    page.cropbox.lower_left = (0, 0)
                    page.cropbox.upper_right = (page_width, pydf2_max_y)
                
                # 裁剪结束页：保留结束关键词之前的内容
                elif i == end_page - 1:
                    page_width, page_height = append_end_info['page_dimensions']
                    _, _, _, max_y = append_end_info['keyword_box']
                    pydf2_max_y = page_height - max_y
                    page.cropbox.lower_left = (0, pydf2_max_y)
                    page.cropbox.upper_right = (page_width, page_height)
                
                append_pages.append(page)
    
    return append_pages
```

## 测试用例

### 测试场景1：正常情况（开始页 < 结束页）

* 前段：找到 start\_info, end\_info

* 后段：找到 append\_start\_info（第N页）, append\_end\_info（第N+2页）

* 预期：追加内容包含第N页（裁剪后）+ 第N+1页（完整）+ 第N+2页（裁剪后）

### 测试场景2：只找到结束关键词

* 前段：找到 start\_info, end\_info

* 后段：只找到 append\_end\_info（第N页）

* 预期：追加内容包含第N-1页（完整）+ 第N页（裁剪后，保留结束关键词之前的内容）

### 测试场景3：只找到开始关键词

* 前段：找到 start\_info, end\_info

* 后段：只找到 append\_start\_info（第N页）

* 预期：追加内容包含第N页（裁剪后，保留开始关键词之后的内容）+ 第N+1页（完整）

### 测试场景4：追加内容关键词都没找到

* 前段：找到 start\_info, end\_info

* 后段：都没找到

* 预期：只输出主内容，不追加

### 测试场景5：开始页 > 结束页（异常情况）

* 前段：找到 start\_info, end\_info

* 后段：找到 append\_start\_info（第N页）, append\_end\_info（第N-1页）

* 预期：按结束关键词页往前取一页处理，只输出第N-2页

### 测试场景6：开始和结束在同一页

* 前段：找到 start\_info, end\_info

* 后段：append\_start\_info 和 append\_end\_info 在同一页

* 预期：裁剪同一页，保留开始关键词之后、结束关键词之前的内容
