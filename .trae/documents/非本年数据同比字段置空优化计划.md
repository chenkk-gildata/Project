# 非本年数据同比字段置空优化计划

## 一、需求背景

### 业务场景

年度报告一般会披露近三年的财务数据。例如2025年年报会披露：

* 2025-12-31（本年度）

* 2024-12-31（上年同期）

* 2023-12-31（上上年同期）

### 问题说明

同比增减字段（`*TBZZ`）只有本年度才会有值，上年同期的同比数据应该为空。但AI提取时可能会错误地为历史年度也提取了同比数据。

### 处理规则

* 按JZRQ降序排序，第一条为本年度数据（保留同比字段）

* 后续为历史年度数据，需要将`*TBZZ`字段置空

## 二、涉及字段

需要置空的同比字段（以`TBZZ`结尾）：

```
YYZSRTBZZ, YYSRTBZZ, KCHYYSRTBZZ, JLRHJTBZZ, JLRTBZZ, KCFJYXSYHDJLRTBZZ, PTGJLRTBZZ, KCFJCXSYHPTGJLRTBZZ
```

## 三、实现方案

### 修改位置

在 `_merge_extracted_data` 方法中，合并完成后添加处理逻辑。

### 实现步骤

1. **定义同比字段列表**

   * 新增常量 `TBZZ_FIELDS`，包含所有同比增减字段

2. **添加处理方法**

   * 新增 `_clear_tbzz_for_historical_data` 方法

   * 按JZRQ降序排序数据

   * 跳过第一条（本年度），处理后续记录

   * 将`*TBZZ`字段置空

3. **调用处理方法**

   * 在 `_merge_extracted_data` 返回前调用处理方法

## 四、代码修改

### 4.1 新增常量

```python
# 同比增减字段列表
TBZZ_FIELDS = [
    "YYZSRTBZZ", "YYSRTBZZ", "KCHYYSRTBZZ", 
    "JLRHJTBZZ", "JLRTBZZ", "KCFJYXSYHDJLRTBZZ", 
    "PTGJLRTBZZ", "KCFJCXSYHPTGJLRTBZZ"
]
```

### 4.2 新增方法

```python
def _clear_tbzz_for_historical_data(self, data: List[Dict[str, Any]]) -> List[Dict[str, Any]]:
    """
    清空历史年度数据的同比增减字段
    
    年度报告披露近三年数据，只有本年度（最新年度）才有同比数据，
    历史年度的同比字段应置空
    
    Args:
        data: 合并后的数据列表
        
    Returns:
        处理后的数据列表
    """
    if not data or len(data) <= 1:
        return data
    
    sorted_data = sorted(data, key=lambda x: x.get("JZRQ", ""), reverse=True)
    
    for i, record in enumerate(sorted_data):
        if i == 0:
            continue
        
        for field in TBZZ_FIELDS:
            if record.get(field):
                record[field] = ""
    
    return sorted_data
```

### 4.3 修改 `_merge_extracted_data` 方法

在返回前调用处理方法：

```python
def _merge_extracted_data(self, module1_data, module2_data):
    # ... 原有合并逻辑 ...
    
    # 清空历史年度的同比字段
    merged_data = self._clear_tbzz_for_historical_data(merged_data)
    
    return merged_data
```

## 五、处理流程图

```
合并模块一和模块二数据
        │
        ▼
  按JZRQ降序排序
        │
        ▼
  ┌─────────────────┐
  │ 遍历每条记录    │
  └─────────────────┘
        │
        ├─ 第一条（i=0）→ 保留原值（本年度数据）
        │
        └─ 后续记录（i>0）→ 将*TBZZ字段置空（历史年度数据）
        │
        ▼
  返回处理后的数据
```

## 六、示例

### 处理前

```json
[
  {"JZRQ": "2025-12-31", "YYSR": "1000", "YYSRTBZZ": "10.5"},
  {"JZRQ": "2024-12-31", "YYSR": "900", "YYSRTBZZ": "8.2"},
  {"JZRQ": "2023-12-31", "YYSR": "830", "YYSRTBZZ": "5.1"}
]
```

### 处理后

```json
[
  {"JZRQ": "2025-12-31", "YYSR": "1000", "YYSRTBZZ": "10.5"},
  {"JZRQ": "2024-12-31", "YYSR": "900", "YYSRTBZZ": ""},
  {"JZRQ": "2023-12-31", "YYSR": "830", "YYSRTBZZ": ""}
]
```

## 七、注意事项

1. **排序依据**：按JZRQ降序排序，最新日期在前
2. **只处理第一条之后的数据**：第一条为本年度数据，保留同比字段
3. **只置空有值的字段**：如果字段本身为空，不做处理
4. **不影响其他字段**：只处理`*TBZZ`字段，其他字段保持不变
