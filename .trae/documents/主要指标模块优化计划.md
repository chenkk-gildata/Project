## 优化点1：追加内容保存位置

### 测试程序实现 (L569-607)

### 主程序当前实现
- `_append_content_to_pdf`: 将追加内容合并到主PDF
- 缺少单独保存追加内容PDF的功能

### 优化步骤
1. 在 `ZyzbProcessor` 类中添加 `_save_append_content_pdf` 方法
2. 将追加内容保存到 `output_dir/净资产收益率和每股收益/` 子目录

4. 修改 `process_pdf` 方法，调用独立保存方法而非合并方法

## 优化点2：目录检测跳过机制

### 测试程序实现 (L109-115)
```python
skip_directory_check = False
for page_num in range(...):
    # 检查是否为目录页，且还需要检查目录
    if not skip_directory_check:
        directory_instances = page.search_for("目录")
        if directory_instances:
            skip_directory_check = True  # 目录页通常只有一页，跳过之后不再检查
            continue
```

### 优化步骤
1. 在 `find_keywords` 方法中添加 `skip_directory_check = False` 变量
2. 在遍历搜索范围时添加目录检测逻辑
3. 找到目录页后设置 `skip_directory_check = True` 跳过后续检查
