# 主要股东背景介绍小程序打包说明

## 打包步骤

### 方法一：使用批处理文件（推荐）

1. 双击运行 `build_app.bat` 文件
2. 等待打包完成
3. 打包完成后，可执行文件将位于 `dist\主要股东背景介绍比对系统.exe`

### 方法二：手动使用PyInstaller

1. 确保已安装PyInstaller和pefile模块：
   ```
   pip install pyinstaller
   pip install pefile==2023.2.7
   ```

2. 打开命令提示符或PowerShell
3. 进入项目目录：
   ```
   cd e:\Project\主要股东背景介绍
   ```
4. 执行打包命令：
   ```
   pyinstaller main_SHBackground.spec
   ```
5. 打包完成后，可执行文件将位于 `dist\主要股东背景介绍比对系统.exe`

## 注意事项

1. **环境要求**：
   - Python 3.8 或更高版本
   - 已安装所有项目依赖（pandas, openai, pyodbc, zhconv, tenacity等）

2. **打包前检查**：
   - 确保所有Python文件没有语法错误
   - 确保所有依赖模块已正确安装
   - 确保配置文件中的API密钥等敏感信息已正确设置

3. **打包后使用**：
   - 可执行文件可以在没有Python环境的Windows系统上运行
   - 确保目标系统可以访问数据库服务器（10.101.0.212）
   - 确保目标系统可以访问AI服务API（https://dashscope.aliyuncs.com）

4. **常见问题**：
   - 如果打包失败，请检查是否所有依赖都已安装
   - 如果运行时出现模块找不到的错误，可能需要在spec文件中添加更多的hiddenimports
   - 如果数据库连接失败，请检查网络连接和数据库服务器状态
   - 如果遇到"ModuleNotFoundError: No module named 'pefile'"错误，请安装兼容版本：`pip install pefile==2023.2.7`

## 文件说明

- `main_SHBackground.py`: 主程序文件
- `main_SHBackground.spec`: PyInstaller配置文件
- `build_app.bat`: 打包批处理文件
- `config.py`: 配置文件
- `database_manager.py`: 数据库管理模块
- `ai_service_enhanced.py`: AI服务模块
- `logger_config.py`: 日志配置模块

## 技术支持

如遇到问题，请检查：
1. Python环境和依赖是否正确安装
2. 网络连接是否正常
3. 数据库和AI服务是否可访问
4. 配置文件中的参数是否正确
