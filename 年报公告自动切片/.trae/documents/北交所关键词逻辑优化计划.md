# 北交所关键词逻辑优化计划

## 背景说明

用户对北交所的关键词逻辑进行了优化：

* 原来北交所没有移除关键词

* 现在北交所也有移除关键词了（第33-39行已添加）

* 原来的结束关键词"变动情况"变成了现在的移除开始关键词

**北交所关键词配置（第33-39行）：**

```python
end_pattern = 股权激励情况
remove_start_pattern = 变动情况  # 原来的结束关键词
remove_end_pattern = 董事...决策程序...实际支付情况
end_keyword = ["股权激励情况"]
remove_start_keyword = ["变动情况"]
remove_end_keyword = ["实际支付"]
```

## 需要修改的位置

### 一、find\_keywords 方法

#### 1. 第139行 - 搜索开始关键词的条件

**原代码：**

```python
if not remove_start_info or (exchange_code == "bjs" and not end_info):
```

**问题：** 原来北交所不需要移除关键词，所以条件特殊。现在北交所也有移除关键词了。
**修改为：**

```python
if not start_info:
```

#### 2. 第150行 - 搜索结束关键词的条件

**原代码：**

```python
if (exchange_code == "bjs" or remove_end_keyword) and not end_info and start_info:
```

**问题：** 原来北交所特殊处理。现在应该统一。
**修改为：**

```python
if not end_info and start_info:
```

#### 3. 第160-180行 - 搜索移除关键词

**原代码：**

```python
# 搜索移除关键词(非北交所)
if exchange_code != 'bjs':
    if start_info and not remove_start_info:
        ...
    if remove_start_info and not remove_end_info:
        ...
```

**问题：** 原来只有非北交所才搜索移除关键词。现在北交所也需要。
**修改为：** 移除 `if exchange_code != 'bjs':` 判断，统一处理

#### 4. 第182-190行 - 检查是否完成

**原代码：**

```python
if exchange_code != 'bjs':
    if start_info and end_info and remove_start_info and remove_end_info:
        found = True
        break
else:
    if start_info and end_info:
        found = True
        break
```

**问题：** 原来北交所不需要移除关键词就能完成。现在需要统一。
**修改为：**

```python
if start_info and end_info and remove_start_info and remove_end_info:
    found = True
    break
```

#### 5. 第197-204行 - 返回结果

**原代码：**

```python
result = {
    'start': start_info,
    'end': end_info,
    'is_bjs': exchange_code == 'bjs'
}
if exchange_code != 'bjs':
    result['remove_start'] = remove_start_info
    result['remove_end'] = remove_end_info
```

**问题：** 原来北交所不返回移除关键词信息。现在需要统一返回。
**修改为：**

```python
result = {
    'start': start_info,
    'end': end_info,
    'remove_start': remove_start_info,
    'remove_end': remove_end_info
}
```

#### 6. 第79行 - get\_search\_rect 关键词列表

**原代码：**

```python
if keyword_type in ['高级管理人员', '履行职责', '履职', '主要工作经历', '变动情况' '有关情况', '高级管理人员离任', '报酬情况', '薪酬情况']:
```

**问题：** 缺少北交所的关键词，且第79行有语法错误（'变动情况' '有关情况'中间缺少逗号）
**修改为：**

```python
if keyword_type in ['高级管理人员', '履行职责', '履职', '主要工作经历', '变动情况', '有关情况', '高级管理人员离任', '报酬情况', '薪酬情况', '股权激励情况', '实际支付']:
```

### 二、process\_pdf 方法

#### 1. 第216行 - 移除is\_bjs变量

**原代码：**

```python
is_bjs = keywords.get('is_bjs', False)
```

**问题：** 不再需要区分北交所
**修改为：** 删除此行

#### 2. 第231行 - 只找到开始关键词时的处理

**原代码：**

```python
pagenum = 2 if is_bjs else 4
```

**问题：** 不再需要区分北交所
**修改为：**

```python
pagenum = 4
```

#### 3. 第249-250行 - 处理移除区域的条件

**原代码：**

```python
if not is_bjs and remove_start_info and remove_end_info:
```

**问题：** 不再需要区分北交所
**修改为：**

```python
if remove_start_info and remove_end_info:
```

#### 4. 第374-410行 - 北交所处理逻辑分支

**原代码：**

```python
else:
    # 北交所PDF处理逻辑(或移除开始or移除结束关键词为空)，需要处理开始结束裁剪页
    ...
```

**问题：** 这个else分支现在不再需要，应该删除。所有交易所都走统一的移除区域处理逻辑。
