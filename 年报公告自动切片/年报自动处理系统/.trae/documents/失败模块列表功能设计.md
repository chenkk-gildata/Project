# 失败模块列表功能设计计划（最终版）

## 需求分析

在状态查看工具中新增选项，查看处理失败的模块列表：
- 股票代码、证券简称、发布日期
- 失败的业务模块名称（无输出、失败、跳过）
- **实时日期筛选功能**（输入日期时数据自动刷新）

## 实时刷新功能说明

### 实现原理

使用 `msvcrt` 模块（Windows）或 `readchar` 库实现非阻塞键盘输入，用户每输入一个字符，终端就实时刷新显示筛选结果。

### 交互流程

```
============================================================
失败模块列表
============================================================

请输入日期进行筛选 (格式: YYYY-MM-DD，按 ESC 退出):
> 2026-03-11

模块失败统计 (2026-03-11):
+------------+------+--------+------+------+------+
| 模块名     | 成功 | 无输出 | 失败 | 跳过 | 总计 |
+------------+------+--------+------+------+------+
| 领导人持股 |   56 |      1 |    0 |    0 |   57 |
+------------+------+--------+------+------+------+

失败详情:
+----------+----------+------------+------------+------------+
| 股票代码 | 证券简称 | 发布日期   | 模块名称   | 失败状态   |
+----------+----------+------------+------------+------------+
| 002668   | TCL智家  | 2026-03-11 | 领导人持股 | 无输出     |
+----------+----------+------------+------------+------------+
```

### 交互特点

1. **实时刷新**：输入日期时，数据自动刷新显示
2. **增量筛选**：输入 `2026` 显示所有2026年的数据，输入 `2026-03` 显示3月数据
3. **支持删除**：按退格键可删除输入，数据实时更新
4. **退出方式**：按 ESC 键返回上级菜单

### 输入行为

| 用户输入 | 显示结果 |
|----------|----------|
| (空) | 显示全部失败记录 |
| `2026` | 显示2026年的失败记录 |
| `2026-03` | 显示2026年3月的失败记录 |
| `2026-03-11` | 显示2026年3月11日的失败记录 |
| 退格键 | 删除最后一个字符，刷新数据 |
| ESC键 | 退出返回上级菜单 |

## 技术实现

### 使用 `readchar` 库（推荐）

跨平台支持，简单易用：

```python
import readchar

def real_time_filter():
    date_input = ""
    while True:
        # 清屏并显示当前筛选结果
        clear_screen()
        display_results(date_input)
        
        # 读取单个字符
        ch = readchar.readchar()
        
        if ch == readchar.key.ESC:
            break
        elif ch == readchar.key.BACKSPACE:
            date_input = date_input[:-1]
        else:
            date_input += ch
```

## SQL 查询逻辑

### 按日期前缀筛选

```sql
SELECT 
    a.gpdm, a.zqjc, a.publish_date, 
    m.module_name, m.status
FROM module_records m
JOIN announcements a ON m.hashcode = a.hashcode
WHERE m.status != 'success' AND a.publish_date LIKE ?
ORDER BY a.publish_date DESC, a.gpdm
```

参数格式：`2026-03%` 或 `2026-03-11%`
