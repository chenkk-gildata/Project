# 日志输出优化方案

## 需求分析

用户希望减少重复的无用日志输出，具体包括：

1. **系统运行统计**：当系统状态没有变化时（没有新公告、没有正在处理的任务），不重复打印统计信息
2. **"没有需要恢复的处理任务"**：连续多次查询都没有需要恢复的任务时，只在第一次打印
3. **"数据库查询完成,发现 0 个新公告"**：连续查询都是0个新公告时，只在第一次打印

## 涉及文件

* `main.py` - `_print_stats()` 方法

* `monitor.py` - `_query_database()` 和 `_check_new_announcements()` 方法

* `task_dispatcher.py` - `_recover_pending_tasks()` 方法

修改 `_print_stats()` 方法：

```python
def _print_stats(self):
    """打印统计信息"""
    while self._running and not self._stop_event.is_set():
        # ... 等待逻辑 ...
        
        # 获取统计信息
        db_stats = db.get_statistics()
        queue_stats = queue_manager.get_all_stats()
        
        # 构建当前状态快照
        current_stats = {
            'total': db_stats.get('total', 0),
            'pending_download': db_stats.get('pending_download', 0),
            'downloading': db_stats.get('downloading', 0),
            'download_success': db_stats.get('download_success', 0),
            'download_failed': db_stats.get('download_failed', 0),
            'pending_process': db_stats.get('pending_process', 0),
            'processing': db_stats.get('processing', 0),
            'process_success': db_stats.get('process_success', 0),
            'process_failed': db_stats.get('process_failed', 0),
            'download_queue': queue_stats['download_queue']['current_size'],
            'process_queue': queue_stats['process_queue']['current_size'],
            'active_download': downloader.get_active_count(),
            'active_process': task_dispatcher.get_active_count(),
        }
        
        # 检查状态是否有变化
        if self._last_stats == current_stats:
            self._stats_unchanged_count += 1
            # 状态未变化，不打印
            continue
        
        # 状态有变化，打印统计信息
        self._last_stats = current_stats
        self._stats_unchanged_count = 0
        
        # ... 打印统计信息 ...
```

```python
def _query_database(self) -> List[Announcement]:
    # ... 查询逻辑 ...
    
    # 更新最后查询时间
    db.set_last_query_time()
    
    # 日志优化：连续查询到0个新公告时，只在第一次打印
    if len(announcements) == 0:
        if self._zero_count_consecutive == 0:
            logger.info(f"数据库查询完成,发现 0 个新公告")
        self._zero_count_consecutive += 1
    else:
        self._zero_count_consecutive = 0
        logger.info(f"数据库查询完成,发现 {len(announcements)} 个新公告")
    
    return announcements
```

* 当系统处于空闲状态（无新公告、无处理任务）时，终端不会重复打印相同的统计信息

* 当有新公告或任务状态变化时，正常打印相关信息

* 用户可以更清晰地查看有用的日志信息
